# Cycle Tracker Card

Card Lovelace pentru monitorizarea ciclului menstrual în Home Assistant.

## Instalare

1. Instalează integrarea [cycle-tracker-ha](https://github.com/simabogdan1519-sketch/cycle-tracker-ha) mai întâi
2. Adaugă acest repo în HACS → Frontend
3. Adaugă cardul în dashboard:
```yaml
type: custom:cycle-tracker-card
```
